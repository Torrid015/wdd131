Site Name

Osoko Odunayo (Torrid) - This name represents my personal portfolio showcasing both my web development projects and Fashion Designing projects. It is simple, memorable, and reflects my technical skills,creativity and brand identity.

Site Purpose

The website serves as a personal portfolio and professional showcase. It provides:

  • Information about my skills and background in Web Development and Fashion Designing.
  • Showcase of completed projects with descriptions and images.
  • A contact form for potential clients or collaborators to reach me.

Scenarios

Questions from a visitor representing the target audience:

  • How can I view Odunayo's completed web development projects?
  • How do I contact Odunayo for a project or collaboration?

Color Schema

The site uses the following primary colors:

  • Primary: #1f344d - Used for headings, logo accents, and footer background.
  • Secondary: #f4f4f4 - Used for page background and section backgrounds.
  • Accent: #004d4d - Used for buttons, links, and highlights.

Typography

Selected fonts for the website:

  • 'Open Sans' – Used for headings (projects, page titles).
  • 'Open Sans' – Used for body text and paragraphs.
  • 'Roboto' – Used for form input and labels for readability.
